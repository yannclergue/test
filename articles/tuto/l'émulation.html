Installez et configurez les 2 meilleurs émulateurs Switch sur PC gratuitement

Vous voulez pouvoir jouer aux jeux Nintendo Switch sortit récemment gratuitement, mais vous ne savez pas comment faire ? Pas de panique, l’émulation est là pour vous sauver. Dans ce tutoriel, je vous montrerai étape par étape comment installez et configurer Ryujinx et Yuzu.

Pourquoi installez 2 émulateurs ?

La question est légitime, « pourquoi prendre plus de place sur mon disque quand un seul émulateur suffit ? ». La réponse est simple, la communauté de l’émulation a beau être grandes, il leur faut du temps pour correctement adapter les jeux sur un émulateur, il peut alors arriver qu’un émulateur puisse mieux émuler un jeu qu’un autre. C’est pour cette raison qu’il devient préférable d’avoir 2 émulateurs aux lieux d’un, si un jeu ne marche pas ou bug chez l’un, je le teste chez l’autre.

Commençons avec Ryujinx

Ryujinx est un émulateur Switch open source gratuit dont le but principal est de fournir la plus belle qualité visuelle de jeux possible. Le projet à commencer en 2017 et il est devenu un émulateur très important qui a su faire sa place.

L’installation

Sur ce, commençons l’installation. Pour pouvoir pleinement profiter de Ryujinx il vous faudra télécharger 3 éléments. Premièrement, Ryujinx lui-même, puis le firmware, qui sont les fichiers permettant de gérer et naviguer dans le système et enfin la prod.key qui permet d’identifier l’installation comme étant une switch.

Les fichiers à télécharger

Lien vers Ryujinx : Lien vers le firmware : Lien vers la prod.key : Attention, seul Window et Linux sont supportés.

Extraction et installation

Une fois les 3 éléments télécharger, commencé par décompresser l’archive contenant Ryujinx, puis lancer Ryujinx.exe. Si une console avec des lignes de codes apparait sur votre écran, cela signifie que l’application est bien lancée. Faite bien attention à ne jamais fermer l’onglet de cette console, sinon l’application se fermera. Au bout d’un certain temps, un message d’erreur va apparaitre. Appuyer simplement sur « ok » et continuer. Une fois Ryujinx lancer, clicker sur « file » en haut à gauche puis sur « Open Ryujinx folder ». Allez dans le dossier « System » et glissé y la « prod.key ». Une fois cela fait, retourner sur l’application et allez cette fois dans « tools » ensuite « install Firmware » et choisissez « Install a firmware from XCI or ZIP », sélectionner ensuite le dossier « firmware.zip » télécharger précédemment. Il ne vous reste qu’à fermer et relancer Ryujinx et c’est bon, vous êtes prêt à jouer.

Configuration et options

Cependant, avant de commencer à utiliser l’émulateur et de joué à vos jeux Switch préféré, il reste une chose importante à faire, configurer l’émulateur pour avoir une agréable expérience de jeu.

Importer ses jeux

Voilà la partie la plus intéressante. Pour trouver les jeux Switch que vous désirez télécharger allez sur ce site : - https://www.emugen.net/ Une fois dessus, chercher le jeu qui vous intéresse et télécharger la base XCI ou la base XCI + update si l’option est disponible. Si des updates et des DLC sont disponibles, je reviendrai plus tard sur la méthode pour les rajouter à votre jeu. Faites cependant attention, le site est entièrement gratuit, mais vous devrez attendre 1h entre chaque téléchargement, commencer donc toujours pas prendre le jeu de base. Une fois le jeu en votre possession, crée un dossier où ranger tous vos jeux et n’oubliez pas de le décompresser. Puis retourner sur Ryujinx, allez dans l’onglet « options » et cliquez sur « Settings ». Ne touchez pas aux cases déjà cochées, allez dans le rectangle en bas et cliquez sur « Add ». Dans la fenêtre qui s’ouvre, sélectionner le dossier contenant vos jeux. Cliquez ensuite sur « Apply » puis « Save ». Maintenant, vos jeux apparaissent dans le menu de Ryujinx. Si vous voulez rajouter un DLC ou une update à l’un de vos jeux, télécharger le, mettez le fichier dans votre répertoire à jeux, décompressez le fichier puis allez sur Ryujinx. Faites un clic droit sur le jeu concerné et choisissez entre « Manage Title Update » ou « Manage DLC » en fonction de votre choix. Dans les 2 cas, dans la fenêtre ouverte, cliquez sur « Add » ensuite sélectionner l’update ou le DLC que vous voulez. N’oubliez pas d’appuyer sur « save ». Une fois la fenêtre fermée, le DLC / l’update est installée.

Les options en jeu

Pour passer votre affichage en plein écran une fois un jeu lancé (pour lancer un jeu, double-cliquez sur son icône) allez dans « Options » et cliquer sur « Enter Full screen ». Si vous voulez automatiquement passer en plein écran à chaque démarrage de jeu, cochez la case juste en dessous « Start game in Fullscreen Mode ». Ensuite, si durant une partie vous souhaitez mettre l’émulation en pause, allez dans l’onglet « Actions » et choisissez « Pause Emulation » puis « Resume Emulation » quand vous désirez reprendre votre jeu. Pour arrêter l’émulation, retournez dans l’onglet « Actions » et cliquez sur « Stop Emulation ». Pour prendre des captures d'écran, allez dans le même onglet et sélectionner « Take Screenshot » ou appuyer sur F8. Vous pouvez retrouver vos captures d'écran dans le dossier « Images » de Windows puis à l’intérieur dans un dossier se nommant « Ryujinx ».

Crée un profil utilisateur

Si vous comptez utiliser l’émulateur à plusieurs personnes, il est préférable de créer un User profile pour chaque personne afin que chacun garde ses sauvegardes et donc sa progression en jeu. Pour créer un User Profile, allez dans « Options » puis « Manage User Profiles ». Dans cette fenêtre, cliquez sur « Add New Profile », commencez par écrire votre pseudo puis validez en cliquant sur « OK ». L’émulateur va ensuite vous demandez si vous préférez importer l’une de vos images comme photo de profil (« Import Image File ») ou si vous désirez prendre une image issue de la switch. Une fois votre image sélectionnée, votre User Profil sera créé. Répétez cette opération autant de fois que vous n’avez de personne différente qui compte jouer via l’émulateur. Le profil sélectionné est celui avec le fond de couleur sur le pseudo. Pour changer d’utilisateur, faite un double-clique sur le profil de votre choix. Enfin, si vous voulez supprimer un profil existant, choisissez-le puis cliquer sur « Delete Selected Profile ». Attention cependant, cette action supprimera toutes les sauvegardes et irréversibles.

Connecter et configurer sa manette

Maintenant que vos jeux sont installés et que votre profil est créé, il reste une étape importante, la configuration de la manette. Pour la configurer, allez dans « Options » puis « Settings » et enfin « Input ». Sur cette page, cliquez sur le bouton « configure » en dessous de « Player 1 ». Dans cette nouvelle fenêtre, allez premièrement en haut à gauche et cliquez sur le rectangle en face de « Input Device » et choisissez votre manette. Si votre manette n’est pas détectée, essayer de relancer l’émulateur en ayant au préalable brancher ou connecté votre manette. Une fois votre manette sélectionner, cliquez sur le bouton « load » se situant vers la droite. Ceci attribuera automatiquement les boutons à votre manette. Évidemment, si vous désirez changer les touches assignées de base, vous pouvez. Pour le reste des options offertes, ne vous en préoccupez pas, votre manette est maintenant fonctionnelle.

Configurer la langue et l’heure

À présent que tout est prêt, il ne reste plus qu'à configuré la langue des jeux en français. Pour ceci, retournez dans « Settings » puis « System ». Commencez par choisir Europe dans la case en face de « System Region » et French en face de « Sytem language ». UTC apparaitra automatiquement en dessous une fois ces 2 paramètres changer. Laissez les autres options comme elle sont et finissez en appuyant sur « Apply » en bas à droite, puis « Save ».

Dernières configurations en pagaille

Concernant les 2 onglets restant dans « Settings » soit « Graphics » et « Logging » laissez les configurations comme t’elle et touchez-y seulement en cas de problème d’affichage ou de bug sur l’émulateur.

Les mods

Un des nombreux avantages qu’offre l’émulation en plus de pouvoir jouer gratuitement à vos jeux est d’offrir du contenu crée par la communauté autrement dit, des mods. Ils peuvent aussi bien apporter des ajouts mineurs tels que des apparences aux personnages ou à l’inverse, rajouter énormément de contenue.

Ou les trouvez

Pour trouver et télécharger les mods qui vous intéressent, je recommande Gamebanana disponible ici. Ce site vous permet de chercher parmi une pléthore de mods répartis chacun pour un jeu en particulier, de quoi changer votre jeu comme vous le désirez.

Comment les installez

L’installation des mods et similaire à celle d’update ou de DLC. Commencez par télécharger un mod qui vous intéresse, extrayez-le puis allez sur Ryujinx. Faite un clic droit sur le jeu que vous souhaitez modé, puis choisissez « Open Mods Directory ». Il ne vous reste plus qu’à placer le dossier de mod dans la fenêtre venant de s’ouvrir, et c’est bon, le mod est prêt. Soyez vigilants avec les notices de certains mods (les notices sont généralement dans la description du mod ou dans un readme.txt dans le dossier après décompression) qui pourront vous indiquez une installation plus complexe pour fonctionne.

Continuons avec Yuzu

Maintenant que Ryujinx est configuré et entièrement jouable, passons à Yuzu. Si Ryujinx à comme objectif de rendre les jeux les plus beaux possibles en dépit parfois de leur stabilité, Yuzu essaie lui, à l’inverse de rendre les jeux les plus stables possibles avant de commencer à les améliorés graphiquement.

L’installation

Pour Yuzu il ne vous faudra télécharger qu’un seul fichier. Prenez de l’avance et récupérer la prod.key utilisé dans Ryujinx, il faudra l’utiliser une nouvelle fois.

Les fichiers à télécharger

Rendez-vous sur le site de Yuzu ici et choisissez votre plateforme pour télécharger le fichier. (Seulement Window et Linux de disponible)

Extraction et installation

Une fois le fichier télécharger, lancer le. Une fenêtre s’ouvre alors avec les options d’installations. Garder les cases déjà cochées et cliquer sur « Install ». Le logiciel va ainsi commencer l’installation. Une fois l’installation terminée, un message d’erreur va apparaitre. Appuyer sur « Ok » afin de fermer la fenêtre. Juste après, une seconde fenêtre vous demandera si vous acceptez que le logiciel recueille des données anonymes pour aider à son amélioration. Décidez vous-même en choisissant « oui » ou « non ». Une fois arrivé dans le menu, allez dans « fichier » puis « Ouvrir le dossier de Yuzu » puis allez dans le dossier « keys ». Une fois dans ce dossier, placez s’y la prod.key téléchargé précédemment.

Configuration

Une fois l’émulateur installé, il faut le configurer pour pouvoir jouer dans les meilleures conditions possibles.

Importer ses jeux

Pour importer vos jeux, faites un double clic en plein milieu du menu de Yuzu puis choisissez votre répertoire. Yuzu et Ryujinx peuvent tout à fait avoir le même répertoire de jeu, cela ne causera aucun problème. Si tout se passe bien, vos jeux devraient apparaitre dans le menu. Pour importer des updates ou des DLC, allez dans l’onglet « fichier » puis cliquer sur « Installez des fichiers sur la NAND ». De la choisissez l’update ou le DLC que vous souhaitez installer. Une fois l’update / DLC choisis, Yuzu vous demandera de confirmer votre choix. Cliquez sur « Installez » et l’opération sera terminée.

Les options en jeu

Une fois dans votre partie, si vous désirez mettre votre jeu en plein écran, allez sur l’onglet « Vue » et sélectionné « Plein écran ». Quant aux commandes pour mettre le jeu en pause ou arrêté l’émulation, elle se trouve dans l’onglet « Emulation ».

Crée un profil utilisateur

Comme pour Ryujinx, il est possible de créer plusieurs profils utilisateur. Pour cela, allez dans l’onglet « Emulation » puis « Configurer », suivis de « Système » (dans la barre à gauche) et enfin dans « profils ». Pour Ajouter un utilisateur, cliquez sur le bouton « Ajouter » situer en bas à droite. Il vous sera alors demandé de choisir un pseudo pour votre utilisateur. Une fois le pseudo remplis, le nouvel utilisateur est créé. Pour choisir entre vos utilisateurs, un simple clic suffit. Si vous souhaitez mettre une image à votre profil, cliquez sur le bouton en bas à gauche « mettre une image » et choisissez l’image que vous souhaitez dans vos dossiers. Enfin, pour supprimer un utilisateur, sélectionner l’utilisateur concerné et cliquer sur « Supprimer » en bas à gauche. Ceci effacera toute les données de sauvegarde liée à l’utilisateur.

Connecter et configurer sa manette

Pour pouvoir configurer votre manette, allez dans « Emulation » puis « Configurez » et enfin « Contrôles » (dans la barre à gauche). Une fois arrivé dans cette fenêtre, commencez par choisir qu’elle type de manette vous voulez émuler dans l’onglet situé juste en dessous de « Connecter la manette » puis sélectionner votre manette dans le rectangle en dessous de « périphérique d’entrée ». Les touches sont maintenant attribuées, il est possible de changer les touches comme vous le désirez. Une fois la configuration terminée, cliquez sur « OK » situé en bas à droite.

Configurer la langue et l’heure

Allez dans « Emulation » puis « Configurer », allez ensuite dans l’onglet « Interface ». Dedans sélectionnez « French » dans le rectangle en face de « Language de l’interface ». Une fois ceci fait, allez dans « Système » (dans la colonne de gauche) puis choisissez comme langue « Français », comme région « Europe » et le fuseau horaire UTC.

Installation de mods

Le téléchargement de mod ce fait sur le même site que Ryujinx. L’installation est-elle aussi assez similaire, faites un clic droit sur le jeu que vous désirez modé, puis cliquez sur « Ouvrir l’emplacement des donnés de mods ». Mettez les dossiers qui vous intéressent dans le dossier qui vient de s’ouvrir et le tour est joué.
